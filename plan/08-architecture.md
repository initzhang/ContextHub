## 核心模块职责

| 模块 | 职责 |
|------|------|
| Context Service | 上下文文件的 CRUD、L0/L1/L2 管理、关系管理 |
| Memory Service | 记忆提取、去重、热度管理、共享/提升 |
| Skill Service | Skill 定义、版本管理、发布/订阅 |
| Retrieval Engine | 意图分析 → 目录层级递归检索 + 关系跳转 → Rerank |
| Indexer | 内容变更时异步生成 L0/L1、更新向量索引 |
| Propagation Engine | 变更事件 → 查依赖方 → 执行 PropagationRule |
| Auth & ACL | 认证、RBAC、资源级权限、字段脱敏 |
| Audit Logger | 操作审计、上下文溯源 |
| Version Manager | Skill 版本、记忆版本（可选） |
| Feedback Collector | 隐式反馈采集、质量评分计算、低质量上下文报告 |
| Lifecycle Manager | 上下文状态机、定期归档/清理、湖表同步删除 |
| Content Store | 内容存储抽象（S3/OSS/LocalFS） |
| Vector Store | 向量索引抽象（Milvus/Qdrant/Chroma） |
| Catalog Connector | 数据目录抽象（Hive/Iceberg/Delta/Mock） |
